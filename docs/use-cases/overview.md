# Overview

Most real-world integrations combine **several MasChain services** together. This
section walks through common business scenarios end-to-end so you can see which
services to use and how they fit.

Each guide follows the same shape:

1. **The problem** — the business need.
2. **Services used** — which MasChain services solve it.
3. **The flow** — the sequence of API calls.
4. **Next steps** — links to the relevant service docs.
