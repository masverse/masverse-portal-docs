Get Wallet Category list​

Retrieves all wallet category records for an organization from the database.

GET

{{base_url}}/api/wallet/wallet-category

HEADERS​

client_id   9b16ae5638534ae1961fb370f874b6cc*

client_secret   sk_9b16ae5638534ae1961fb370f874b6cc*

    "result": [

            "id": 1,
            "organisation_id": 1,
